📂 ОСНОВНЫЕ МОДУЛИ

═══════════════════════════════════════════════════════════════

## database/ - Работа с данными

models.py:
  ✓ User - Пользователи с рейтингом
  ✓ Listing - Объявления о заданиях
  ✓ Chat - Чаты между участниками
  ✓ Message - Сообщения в чатах
  ✓ Rating - Рейтинги и отзывы
  ✓ Complaint - Жалобы на пользователей
  ✓ Transaction - Финансовые операции
  ✓ PaymentMethod - Способы оплаты

db.py:
  ✓ init_db() - Инициализация БД
  ✓ get_db() - Получить сессию
  ✓ get_db_session() - Context manager


## handlers/ - Обработка команд

commands.py:
  ✓ /start - Главное меню
  ✓ create_listing - Создание объявления
  ✓ view_listings - Список объявлений
  ✓ my_profile - Профиль пользователя

chat.py:
  ✓ Система чатов между участниками
  ✓ Отправка сообщений
  ✓ Завершение заданий
  ✓ Система рейтинговки
  ✓ Система жалоб

payment.py:
  ✓ Интеграция с Telegram Stars
  ✓ Управление платежами

admin.py (🆕):
  ✓ /admin - Админ-панель
  ✓ Статистика системы
  ✓ Управление жалобами
  ✓ Управление пользователями
  ✓ Блокировка/разблокировка
  ✓ Управление объявлениями


## utils/ - Вспомогательные функции

database_service.py:
  ✓ DatabaseService - Полный сервис БД
  ✓ CRUD операции для всех моделей

helpers.py:
  ✓ Форматирование дат и цен
  ✓ Проверка валидности
  ✓ Форматирование рейтинга

🚀 БЫСТРЫЙ СТАРТ

═══════════════════════════════════════════════════════════════

Шаг 1: УСТАНОВКА
$ python install.py

Шаг 2: КОНФИГУРАЦИЯ
Отредактируйте .env:
  BOT_TOKEN=123456789:ABCDEFGhij...
  SUPER_ADMIN_ID=987654321
  ADMIN_IDS=987654321,111111111

Шаг 3: ЗАПУСК
$ python run.py     (с проверками)
или
$ python main.py    (прямой запуск)

Шаг 4: АДМИНИСТРИРОВАНИЕ
Отправьте боту: /admin
