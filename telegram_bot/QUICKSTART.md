# Быстрая справка по Telegram Bot

## 🚀 Запуск

### Первый раз
```bash
python install.py
```
Это установит все зависимости и создаст `.env` файл.

### Обычный запуск
```bash
python run.py
```
или
```bash
python main.py
```

## ⚙️ Настройка

### 1. BOT_TOKEN
```bash
# Откройте Telegram
# Найдите @BotFather
# Отправьте: /newbot
# Скопируйте токен в .env файл
```

### 2. Администраторы
```bash
# Отредактируйте .env
# Добавьте:
SUPER_ADMIN_ID=your_id
ADMIN_IDS=id1,id2,id3
```

## 📋 Основные команды

| Команда | Что делает |
|---------|-----------|
| `/start` | Главное меню |
| `/admin` | Админ-панель (только админы) |

## 🔧 Скрипты

| Скрипт | Описание |
|--------|---------|
| `install.py` | Устанавливает зависимости |
| `run.py` | Запускает бота с проверками |
| `main.py` | Основная программа бота |

## 📞 Помощь

### Проблема: Бот не запускается
```bash
# Проверьте Python версию
python --version

# Переустановите зависимости
python install.py

# Запустите с отладкой
python main.py
```

### Проблема: BOT_TOKEN ошибка
```
# Убедитесь что в .env правильный токен
# Откройте .env и проверьте BOT_TOKEN=...
```

### Проблема: Админ панель не работает
```
# Убедитесь что вы администратор
# Проверьте .env - ваш ID в ADMIN_IDS или SUPER_ADMIN_ID
# Перезагрузите бота
```

## 💡 Советы

- 💾 Регулярно делайте резервные копии `bot_database.db`
- 🔐 Не делитесь своим BOT_TOKEN
- 👤 Используйте `@userinfobot` для получения своего ID
- 📊 Проверяйте админ-панель регулярно
