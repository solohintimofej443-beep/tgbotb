# Telegram Bots Task Platform
Платформа для создания анонимных заданий с системой оплаты через Telegram Stars.

## ⚡ Быстрый старт

```bash
# Автоматическая установка зависимостей
python install.py

# Отредактируйте .env и добавьте BOT_TOKEN

# Запуск бота
python main.py
```

### 1. Создание объявлений
- Пользователь может создавать объявления с названием, описанием и ценой
- Возможность сделать объявление анонимным
- Выбор способа оплаты: предоплата, сразу, после выполнения
- Стоимость публикации: 10 звёзд

### 2. Система оплаты
- Интеграция с Telegram Stars
- Поддержка различных способов оплаты
- История транзакций
- Система кошелька

### 3. Откл иноклики и чаты
- Исполнители могут откликаться на объявления
- Приватный чат между заказчиком и исполнителем
- История сообщений

### 4. Система рейтинга
- Заказчик оценивает исполнителя (от 1 до 5 звёзд)
- Исполнитель оценивает заказчика
- Хранение отзывов и комментариев
- Отображение среднего рейтинга

### 5. Система жалоб
- Возможность пожаловаться на пользователя
- 3 жалобы = блокировка в системе
- Система разрешения жалоб (для администраторов)

## Установка

### Метод 1: Автоматическая установка (Рекомендуется)

```bash
python install.py
```

Скрипт автоматически:
- ✅ Создаст файл `.env`
- ✅ Установит все зависимости
- ✅ Проверит ваше окружение

### Метод 2: Ручная установка

1. Клонируйте проект:
```bash
cd telegram_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env`:
```bash
cp .env.example .env
# отредактируйте .env
```

4. Запустите бота:
```bash
python main.py
```

## Команды бота

- `/start` - Начало работы с ботом
- `/admin` - Администраторская панель (только для администраторов)
- `📝 Создать объявление` - Создать новое объявление
- `📋 Список объявлений` - Просмотр всех объявлений
- `💬 Мои чаты` - Просмотр активных чатов
- `⭐ Мой профиль` - Просмотр профиля и рейтинга

## Модели данных

### User
- `user_id` - ID пользователя в Telegram
- `rating_as_customer` - Рейтинг как заказчик
- `rating_as_performer` - Рейтинг как исполнитель
- `complaints_count` - Количество жалоб
- `is_blocked` - Флаг блокировки
- `stars_balance` - Баланс звёзд

### Listing
- `id` - ID объявления
- `customer_id` - ID заказчика
- `title` - Название задания
- `description` - Описание
- `price` - Цена в звёздах
- `is_anonymous` - Анонимное ли объявление
- `payment_method` - Способ оплаты
- `status` - Статус (active, in_progress, completed, cancelled)
- `performer_id` - ID исполнителя

### Chat
- `id` - ID чата
- `listing_id` - ID объявления
- `customer_id` - ID заказчика
- `performer_id` - ID исполнителя

### Rating
- `id` - ID рейтинга
- `listing_id` - ID объявления
- `rater_id` - Кто оставляет рейтинг
- `rated_user_id` - Кому оставляют рейтинг
- `score` - Оценка (1-5)
- `rating_type` - Тип ('customer' или 'performer')

### Complaint
- `id` - ID жалобы
- `listing_id` - ID объявления
- `complainant_id` - Кто подал жалобу
- `complained_user_id` - На кого жалоба
- `reason` - Причина жалобы
- `status` - Статус жалобы

## Способы оплаты

1. **Предоплата** - Заказчик платит перед выполнением, исполнитель получает после завершения
2. **Сразу** - Оба платят одновременно в момент завершения чата
3. **После выполнения** - Исполнитель выполняет работу и потом получает оплату

## Система рейтинга

- Минимум: 0 звёзд
- Максимум: 5 звёзд
- Рейтинг считается автоматически как среднее значение всех оценок
- Отображается как оценка и количество отзывов

## Безопасность

- Система жалоб предотвращает мошенничество
- Плохая репутация = блокировка
- Приватные чаты между участниками
- История всех транзакций
