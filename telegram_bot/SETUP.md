# Инструкция по запуску Telegram бота

## Быстрый старт (Все просто!)

### Шаг 1: Автоматическая инициализация

```bash
python install.py
```

Этот скрипт автоматически:
- Проверит версию Python (нужна 3.9+)
- Установит все зависимости
- Создаст файл `.env` (если его нет)
- Проверит все импорты

## Решение проблем

### Бот не отвечает
- Проверьте токен в `.env` файле
- Убедитесь, что интернет-соединение активно
- Перезагрузите бота: остановите и заново запустите `python main.py`

### Ошибки БД
```bash
# Удалить старую БД (потеряются все данные!)
rm bot_database.db

# Затем запустите бота повторно
python main.py
```

### Ошибка импорта модулей
```bash
# Переустановите зависимости
pip install --upgrade -r requirements.txt
```

## Функции для разработчиков

### Добавление новой команды

Создайте функцию-обработчик в файле `handlers/commands.py`:

```python
async def my_command(update: Update, context: ContextTypes.DEFAULT_TYPE):
    """Описание команды"""
    await update.message.reply_text("Ответ")
```

Затем добавьте обработчик в `main.py`:

```python
application.add_handler(CommandHandler('mycommand', my_command))
```

### Работа с БД

```python
from database.db import SessionLocal
from utils.database_service import DatabaseService

db = SessionLocal()
service = DatabaseService(db)

# Получить пользователя
user = service.get_user(user_id)

# Создать объявление
listing = service.create_listing(
    customer_id=user_id,
    title="Название",
    description="Описание",
    price=100,
    is_anonymous=False,
    payment_method="prepayment"
)

db.close()
```
