## 🚨 Часто задаваемые вопросы

**Q: С чего начать?**
A: Запустите `python install.py`, потом `python run.py`

**Q: Как получить BOT_TOKEN?**
A: Найдите @BotFather в Telegram, отправьте /newbot

**Q: Какая версия Python нужна?**
A: Python 3.9 или выше

**Q: Как стать администратором?**
A: Добавьте свой ID в `.env` файл в переменную ADMIN_IDS

**Q: Где найти свой Telegram ID?**
A: Напишите боту @userinfobot

**Q: Как удалить данные бота?**
A: Удалите файл `bot_database.db`

**Q: Безопасен ли при публикации мой BOT_TOKEN?**
A: НЕТ! Никогда не публикуйте BOT_TOKEN. Он в .gitignore
