## 🔍 Как использовать

### 1️⃣ Первая установка
```bash
python install.py
```

### 2️⃣ Конфигурация
Отредактируйте `.env`:
```env

SUPER_ADMIN_ID=987654321
ADMIN_IDS=987654321,111111111
```

### 3️⃣ Запуск
```bash
python run.py
# или
python main.py
```

### 4️⃣ Администрирование
Отправьте боту: `/admin`
