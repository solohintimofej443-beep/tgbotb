# 🚀 НАЧНИТЕ ОТСЮДА

## Быструю старт в 3 шага

### Шаг 1️⃣ : Установка (из корневой папки)

```bash
python setup.py
```

Или если вы в папке `telegram_bot`:

```bash
python install.py
```

### Шаг 2️⃣ : Конфигурация

Отредактируйте файл `telegram_bot/.env`:

```env
# Замените your_token_here на реальный токен
BOT_TOKEN=123456789:ABCDEFGhijklmnopqrstuvwxyz

# Замените на свой ID (получить у @userinfobot)
SUPER_ADMIN_ID=987654321

# ID администраторов (опционально)
ADMIN_IDS=987654321,111111111
```

### Шаг 3️⃣ : Запуск бота

#### Вариант А (рекомендуется):
```bash
cd telegram_bot
python run.py
```

#### Вариант Б:
```bash
python telegram_bot/run.py
```

## 🆘 Если что-то не работает

### Проблема: Python не найден
```bash
# Убедитесь что Python установлен и добавлен в PATH
python --version
```

### Проблема: Файл .env не создаётся
```bash
# Убедитесь что запускаете из правильной директории
cd telegram_bot
python install.py
```

### Проблема: requirements.txt не найден
```bash
# Убедитесь что вы в папке telegram_bot
# Если запускаете из корневой папки, используйте setup.py:
python setup.py
```

## 💡 Советы

- 🔐 Никогда не публикуйте BOT_TOKEN
- 📁 Всегда запускайте из папки `telegram_bot` или используйте `setup.py`
- 📱 Получить ID: напишите боту `@userinfobot`
- 🤖 Получить токен: напишите `@BotFather` и отправьте `/newbot`
